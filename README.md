# 🌍 Global Cybersecurity Attack Analysis Dashboard

**A Power BI Capstone Project**  
**MS Elevate AICTE Internship / Capstone Project**  
**Presented by:** Rahul Bardhan

## 📌 Project Overview

This project presents an **interactive Power BI dashboard for analyzing global cybersecurity threats from 2015–2024**.

The dashboard transforms raw cybersecurity data into **visual insights**, helping identify attack trends, affected industries, geographical patterns, and financial impacts of cyberattacks.

The project demonstrates the use of **data visualization, business intelligence, and cybersecurity analytics** to support informed decision-making.


## 🎯 Objectives

- Analyze global cybersecurity attack data
- Identify **major threat types and attack trends**
- Determine **most affected industries and countries**
- Visualize **financial impact of cyberattacks**
- Build an **interactive business intelligence dashboard**


## 📊 Dataset

Dataset used in this project:

🔗 **Global Cybersecurity Threats (2015–2024)**  
https://www.kaggle.com/datasets/atharvasoundankar/global-cybersecurity-threats-2015-2024

Dataset includes:

- Country
- Year
- Threat Type
- Attack Vector
- Affected Industry
- Financial Impact ($ Millions)
- Severity Level
- Response Time (Hours)


## 🖥️ Dashboard Pages

### 1️⃣ Overview Dashboard
Provides a high-level summary of cybersecurity attacks.

Key metrics include:

- Total cyberattacks
- Total financial loss
- Percentage of critical attacks
- Year-wise attack trends


### 2️⃣ Geographical Insights

Visualizes global cyberattack distribution using **map-based visualizations**.

Highlights:

- Countries with highest attack frequency
- Regional cybersecurity patterns



### 3️⃣ Industry Impact Analysis

Analyzes industries most affected by cyberattacks.

Most targeted industries include:

- Finance
- Healthcare
- Government
- Technology


### 4️⃣ Attack Types & Vulnerabilities

Breakdown of cyberattack categories such as:

- Phishing
- Ransomware
- Man-in-the-Middle
- Malware


## 📈 Key Insights

- **Total Attacks:** 2500+
- **Total Financial Loss:** $889.6 Billion
- **Top Threat Types:** Phishing, Ransomware, Man-in-the-Middle
- **Most Affected Countries:** USA, China, India
- **Most Targeted Industries:** Finance & Healthcare
- **Average Response Time:** ~45 Hours
